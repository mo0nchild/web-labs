Cascading Style Sheets

Cascading Style Sheets (Таблицы Каскадных стилей) - это язык, содержащий набор свойств для описания внешнего вида любых HTML документов.

С его поимощью дизайнер имеет полный контроль над стилем и расположением каждого элемента веб страницы.

Это проще и гораздо функциональнее использования обычного набора HTML тегов.
